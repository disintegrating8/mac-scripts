## Before Running the Script
- install homebrew

## Installation

1. Clone this repository:

```bash
git clone https://github.com/disintegrating8/mac-scripts.git
```

2. go to mac-scripts directory:

```bash
cd mac-scripts
```

3. make it executable:

```bash
chmod +x run.sh
```

4. Run the setup script:

```bash
./run.sh
```
