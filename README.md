## Before Running the Script
- install homebrew

## Installation

1. Clone this repository:

```bash
git clone https://github.com/disintegrating8/mac-scripts.git
```

2. Run the setup script:

```bash
./run.sh
```

3. Only setup tiling environment

```bash
./tiling.sh
```

4. Only setup terminal

```bash
./terminal.zsh
```
